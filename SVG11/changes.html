付録 Q: 変更点

この付録は参考であって、規定ではない。

SVG 1.1 第２版 勧告案 からの変更点は、次のもののみである： The only changes made to the document since the SVG 1.1 Second Edition Proposed Recommendation were the following:

  • 付録：参照文献 の中の CSS 2.0 への参照において、 SVG 仕様の将来版が CSS 2.1 および CSS Fonts Module Level 3 を参照することになる旨を追記。 Some additional wording in the CSS 2.0 reference in the References Appendix to indicate that future versions of the SVG specification will reference CSS 2.1 and CSS Fonts Module Level 3.
  • 参考文献の MathML への参照を MathML 2.0 から MathML 3.0 に更新。 An update to the informative MathML reference from MathML 2.0 to MathML 3.0.
